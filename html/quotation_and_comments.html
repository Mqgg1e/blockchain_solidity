HTML Quotation and Citation Elements.

Element Description Example
<abbr> Defines an abbreviation or acronym WHO
<address> Defines contact information for the author/owner of a document
123 Main St, City, Country
<bdo> Defines the text direction This text is right-to-left
<blockquote> Defines a section that is quoted from another source
This is a blockquote.
<cite> Defines the title of a work The Great Gatsby
<q> Defines a short inline quotation This is a short quote.
